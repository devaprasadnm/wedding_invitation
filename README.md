# Wedding Invitation Platform

A production-ready, mobile-first wedding invitation platform featuring a FastAPI backend, React frontend, and Supabase integration.

## Architecture

- **Backend**: FastAPI (Python)
- **Frontend**: React (Vite + TypeScript)
- **Database**: PostgreSQL (Supabase)
- **Object Storage**: Supabase Storage
- **Deployment**: Dockerized (Cloud Run ready)

## Project Structure

- `/backend`: Python FastAPI application
- `/frontend`: React SPA application
- `docker-compose.yml`: Local development orchestration

## Getting Started

See individual `README.md` files in `backend/` and `frontend/` for specific instructions.
